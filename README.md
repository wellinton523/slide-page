# Slide Page

Site: https://wellinton523.github.io/slide-page/

# **Manual de utilização**

___teclado:___
```
Setinhas para direita e esquerda, troca de slide anterios e proximo

Espaço proximo slide
```

___Botoes:___

```
Superior direito (Shader leve/ Shader completo):
>Troca a shader utilizada para melhor desenpenho (use se estiver com travamentos)

Inferior direito (Olho):
>Oculta outros botoes

Infeiror meio:
>Controle dos slides e visualização do slide atual

Inferior esquerdo:
>Troca o slide mostrado na pagina ou o tema
```
